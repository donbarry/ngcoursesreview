Need Help

This section will help you understand the functionality and features of application

Display Reviews

Once you logs in with a registered accout you can view reviews given by other people about the course. You need to select a course from the dropdown list and then click on get revies the reviews will be displayed in order in which they were added.

Add Reviews

Add reviews allows you to share your own opinions on the course, you need to select a course from the list of courses. Then you need to write title of your review, title of book and body of review. Once you added your review, the review will be visible under the Display review section immediately.

Books

This section will help you find the recommended book for the course and price of the book. Each course has a book associated with it which is suggested by the department. First select the course from dropdown and click on books, this will display image, title and price of the book. All this information is given by ebay API.

Register and Login

User need to register with a valid email id and password no other information is needed. Once registered, user can log into the account with these credentials and can get above services. With the registered id you only get access to the application not to the api, if you want to get key for api contact developers directly.

GitHub Link

API

We also offer the API, you can consume that api in your own website and get services like adding and getting reviews. The user need to havean API key for that. In order to get API key contact us directly
